KB Additions Beyond Textbook (4)
Module 2
The 'Chance Line' module provides a dedicated concrete CPA stage before the abstract probability scale. The textbook moves directly from Exploration 4.1 (subjective likelihood) to Definisi 4.1 (formal vocabulary) without a concrete number-line anchoring stage. KB visual model VIR3 (Probability Number Line) and the CPA progression principle (concrete → pictorial → abstract) require this intermediate stage. Without it, students encounter the 0–1 scale as an abstract symbol before they have a physical/visual referent.
Module 13
The 'Reading Convergence Graphs' module is a scaffold required by the textbook's own exercises. Exploration 4.8 and Latihan B questions 5 and 7 require students to read convergence graphs, but the textbook does not explicitly teach graph-reading skills (x-axis = trials, y-axis = relative frequency, identifying the stabilised value). KB evidence shows that graph literacy is a prerequisite skill that must be explicitly taught before it is applied. Without this module, students cannot access the convergence concept in M14.
Module 24
The 'Reading Dot Plots' module is a scaffold required by Section D exercises. Latihan D and Uji Kompetensi questions 23–24 require students to read dot plots to evaluate representativeness, but the textbook introduces dot plots (Definisi, Gambar 4.34, 4.37) without a dedicated practice module for the reading skill. KB evidence (VIR23, Line Plot/Dot Plot visual model) supports explicit instruction in dot plot literacy before application.
Module 20
The 'Practice: Expected Frequency' module is required by the TGAR rhythm principle. The textbook moves directly from Contoh 4.7 (worked example) to Latihan C (exercises) without a structured practice consolidation stage. KB sequencing rules require a Practice module after each Learn module cluster to consolidate before the Challenge module. Without M20, students move from a single worked example directly to challenge-level problems.
