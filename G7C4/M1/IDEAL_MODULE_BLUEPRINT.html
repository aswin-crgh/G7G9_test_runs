G7C4M1 — Learn
Identify a changing quantity in a growing pattern and represent it using a letter (variable), distinguishing it from fixed quantities (constants), and write the algebraic form using the notation 3n.

Abstract Slides 14, 15, 16, 17
This is a concrete-phase Learn module. The progression runs: Concrete observation (Slides 1–4) → Concrete formalisation with vocabulary (Slides 5–6) → Pictorial/Transition bridge via Function Machine (Slide 7) → Concrete derivation of full expression (Slides 8–9) → Concrete interactive practice (Slides 10–11) → Concrete misconception confrontation (Slides 12–13) → Abstract choral recall and summary (Slides 14–15) → Abstract transfer and exit (Slides 16–17). The module does not reach full Abstract phase — it ends at the boundary of concrete-to-abstract, consistent with the primary CPA phase designation of 'concrete'. Abstract slides (14–17) are summary and exit only, not new abstract instruction.

Convention Requirements (4)
Implicit multiplication in 3n
Show '3 × n = 3n' side by side in the Math_Vault slide (Slide 6). Use colour-coding: the multiplier 3 in one colour, the variable n in another. State explicitly: 'In algebra, we drop the × sign when a number multiplies a letter.' Reinforce this in every worked example and in the Exit_Question reveal.
Slide 6
Variable n as 'any value' vs. 'the unknown value'
At the Snapshot moment (Slide 5, Math_Vault), explicitly state: 'In a pattern, n tells us which figure we are looking at — it can be 1, 2, 3, or any number. We are not trying to find n; we are using n to describe the rule.' Avoid the word 'unknown' in all student-facing content in this module. Reinforce in the Math_Trap for M252 (Slide 12).
Slide 5
Algebraic form vs. algebraic expression
Treat 'algebraic form' as the primary student-facing term throughout this module, matching Exploration 4.1 language. On the Math_Vault slide (Slide 5), note in parentheses: 'also called algebraic expression — they mean the same thing.' Do not introduce 'algebraic expression' as a separate concept; that disambiguation belongs to G7C4M2.
Slide 5
Constant '1' in 1 + 3n — fixed quantity from the pattern
Use the fixed-vs-changing table from Exploration 4.1 explicitly (Slide 4). Label the columns 'Fixed quantity (constant)' and 'Changing quantity (variable part)'. Colour-code the constant 1 in orange and the variable term 3n in blue in every representation throughout the module. Make the structural reason explicit: '1 is a constant because the fixed corner matchstick never changes as the pattern grows.'

Translation Notes (19)
Localisation metadata from module research — bilingual vocabulary references. Not scored; not a delivery mandate.
Primary TermNative TermLanguageTypeContext
variable variabel Indonesian vocabulary Core algebraic term; Indonesian form is a direct cognate of the English.
constant konstanta Indonesian vocabulary A fixed numerical value in an algebraic expression with no variable; Indonesian form is a direct cognate.
algebraic form bentuk aljabar Indonesian vocabulary A mathematical phrase combining variables, coefficients, and constants; Indonesian uses 'bentuk aljabar' (literally 'algebraic form'). This is the primary term used in Kurikulum Merdeka textbooks including Exploration 4.1.
coefficient koefisien Indonesian vocabulary The numerical multiplier of a variable in an algebraic term; Indonesian form is a direct cognate. Note: 'coefficient' is out of scope for this module (owned by G7C4M2) but the translation pair is carried forward for downstream consumers.
term suku Indonesian vocabulary A single component of an algebraic expression separated by + or −; Indonesian uses 'suku' (literally 'part/member'), not a cognate. Note: 'term' is out of scope for this module (owned by G7C4M2) but carried forward for downstream consumers.
n n Indonesian notation The letter used as a variable to represent the pattern number; notation is identical in both languages. Indonesian textbooks use 'n' as the standard pattern-position variable in Kurikulum Merdeka Grade 7 algebra.
3n 3n Indonesian notation Shorthand for 3 × n; the multiplication sign is omitted when a number multiplies a variable. Notation is identical in both languages; the convention of dropping the × sign must be explicitly taught in both delivery contexts.
algebraic expression bentuk aljabar Indonesian vocabulary Synonym for 'algebraic form' used in the chapter; Indonesian uses the same term 'bentuk aljabar' for both. Not the primary term for this module but carried forward for downstream consumers.
substitution substitusi Indonesian vocabulary Replacing a variable with a specific numerical value to evaluate an expression; Indonesian form is a direct cognate. Out of scope for this module (owned by G7C4M4) but carried forward.
like terms suku sejenis Indonesian vocabulary Terms with identical variable parts; Indonesian uses 'suku sejenis' (literally 'terms of the same type'). Out of scope for this module but carried forward.
unlike terms suku tidak sejenis Indonesian vocabulary Terms with different variable parts; Indonesian uses 'suku tidak sejenis'. Out of scope for this module but carried forward.
collect like terms menyederhanakan suku sejenis Indonesian convention The process of combining like terms to simplify an expression; Indonesian textbooks typically use 'menyederhanakan' (simplify) rather than a direct translation of 'collect'. Out of scope for this module but carried forward.
distributive property sifat distributif Indonesian vocabulary The property that a(b + c) = ab + ac; Indonesian uses 'sifat distributif'. Out of scope for this module but carried forward.
commutative property sifat komutatif Indonesian vocabulary The property that a + b = b + a; Indonesian uses 'sifat komutatif'. Out of scope for this module but carried forward.
associative property sifat asosiatif Indonesian vocabulary The property that (a + b) + c = a + (b + c); Indonesian uses 'sifat asosiatif'. Out of scope for this module but carried forward.
equivalence kesetaraan Indonesian vocabulary The state of two algebraic expressions having the same value for all variable values; Indonesian uses 'kesetaraan' or 'ekuivalen'. Out of scope for this module but carried forward.
expand menjabarkan Indonesian vocabulary To apply the distributive property to remove brackets; Indonesian uses 'menjabarkan'. Out of scope for this module but carried forward.
factor memfaktorkan Indonesian vocabulary To rewrite an expression as a product of its factors; Indonesian uses 'memfaktorkan'. Out of scope for this module but carried forward.
degree derajat Indonesian vocabulary The highest exponent of the variable in a polynomial; Indonesian uses 'derajat', which also means angle degree and academic degree — same polysemy risk as in English. Out of scope for this module but carried forward.
